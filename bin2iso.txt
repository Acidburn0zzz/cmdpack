bin2iso(1)
==========
:doctype: manpage

NAME
----
bin2iso - Convert CD .BIN to .ISO

SYNOPSIS
--------
*bin2iso* ['-f'] 'input_file' 'output_file'

DESCRIPTION
-----------
Converts CD-ROM .BIN images (including full 2352-byte sectors)
into .ISO format (2048-byte sectors).  Automatically detects mode 1
or 2. Will warn if there are any form 2 sectors, which are not
convertable.

OPTION
------
*-f*:: Force conversion and bypass checks
