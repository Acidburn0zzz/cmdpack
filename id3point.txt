id3point(1)
===========
:doctype: manpage

NAME
----
id3point - Pointless ID3v1 tagger

SYNOPSIS
--------
*id3point* ['options'] 'mp3file'...

DESCRIPTION
-----------
A basic ID3v1 tagger for MP3 files, probably superseded by newer
tools, but included here in cmdpack nonetheless.

Only ID3v1 tags are supported.  The track number increments
automatically for each additional file.  The '-tf' option will turn
``Artist - Title.mp3'' into ``Title'', but be sure to check the titles
by hand in case it gets confused.

OPTIONS
-------
*-t TITLE*:: Set ID3 title
*-a ARTIST*:: Set ID3 artist
*-m ALBUM*:: Set ID3 album
*-y YEAR*:: Set ID3 year
*-c COMMENT*:: Set ID3 comment
*-k TRACK*:: Set ID3 track number
*-g GENRE*:: Set ID3 genre number

*-t-*:: Clear ID3 title
*-a-*:: Clear ID3 artist
*-m-*:: Clear ID3 album
*-y-*:: Clear ID3 year
*-c-*:: Clear ID3 comment
*-k-*:: Clear ID3 track number
*-g-*:: Clear ID3 genre number

*-tf*:: Set ID3 title baed on the file name
