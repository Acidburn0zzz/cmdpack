zerofill(1)
===========
:doctype: manpage

NAME
----
zerofill - Create a large, empty file

SYNOPSIS
--------
*zerofill* ['-f'] ['-q'] 'size' 'FILE'

DESCRIPTION
-----------
Creates a file filled with bytes of value zero.  Similar to ``dd
if=/dev/zero'', except there's a progress indicator.

Inside a virtual machine, zerofill can be used to create a large dummy
file in order to zero out all the unused sectors before compressing
the drive image.

'size' may be specified in C style ``0x123'' or ``0123'' notation for
hexadecimal or octal.

OPTIONS
-------
*-f*:: Force overwrite
*-q:: Quiet
